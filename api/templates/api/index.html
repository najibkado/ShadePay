Accept Card Payments

This api lets you accept payments with credit cards.

-X POST

-H "Authorization: Bearer YOUR_SECRET_KEY"

Sample POST data

wallet: your wallet address - String

amount: amount - String NB: Amount will be converted to decimal, if it contains any value other than int float or decimal, it will be result into getting an error

description: description of payment - String

status: initiated - String

ip: your customer ip address - IpV4

returnUrl: your return url - String

Sample Response

You will recieve a response like below if there is no error. Proceed to redirect your user to the checkout url to complete payment

On finish

You will get a POST request to your return url with .successful alongside transaction data.

If transaction is successful or .unsuccessful if transaction is unsuccessful alongside transaction data and error message.

Accept Shadepay payments

This api lets you accept payments with shadepay internal payment services.

-X POST

-H "Authorization: Bearer YOUR_SECRET_KEY"

Sample POST data

wallet: your wallet address - String

amount: amount - String NB: Amount will be converted to decimal, if it contains any value other than int float or decimal, it will be result into getting an error

products: an array of JSON objects containig products details

product JSON objects: {"item": "item name", "amount": "amount", "quantity": "quantity"}

products_description: an array of STRING/STRINGS containig description of each products

product_description JSON objects: {"item": "item name", "description": "description"}

status: initiated - String

returnUrl: your return url - String

Pricing

Max charges: NGN 3,500

Transaction endpoint cost 1.25% + NGN 100.
